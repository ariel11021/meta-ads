優化建議
1

建立 Meta Pixel

Meta Pixel 是一段追蹤碼，裝在你的商品網站後，Meta 就能追蹤「誰看了廣告 → 誰真的買了」，並自動計算 ROAS。

Meta 廣告後台 → 活動管理工具 → 資料來源 → Pixel → 建立 Pixel
  • 命名後按「建立」，你會拿到一串 Pixel ID（15 位數字）
  • 選擇安裝方式：如果有 Shopify / Wix / Shopline 等平台，選「合作夥伴整合」最簡單
2

把 Pixel 安裝到商品網站

依你的電商平台選擇對應方式：

Shopify 後台 → 線上商店 → 偏好設定 → Facebook Pixel ID 貼上即可
Shopline 後台 → 設定 → 行銷工具 → Meta Pixel → 貼上 ID
自架網站 把以下程式碼貼在 <head> 內：
<script> !function(f,b,e,v,n,t,s) ... fbq('init', '你的Pixel ID'); fbq('track', 'PageView'); </script>

完整程式碼在 Pixel 建立後的「手動安裝」頁面可以複製。

3

設定「購買」轉換事件

Pixel 裝好後，還需要告訴 Meta「這個頁面 = 購買完成」，它才會記錄 ROAS。

  • Shopify / Shopline 平台：購買事件通常自動追蹤，無需額外設定
  • 自架網站：在訂單完成頁（感謝頁）加上：
fbq('track', 'Purchase', {value: 訂單金額, currency: 'TWD'});

設定後在 Meta 後台 → 活動管理工具 → 測試事件，可以即時確認是否偵測到。

4

把廣告連結從 Portaly 改成直接連到商品頁

這是最關鍵的一步。現在的流程是：

Meta 廣告 → Portaly（中繼頁）→ 商品頁

改成：

Meta 廣告 → 商品頁（直接）
  • 進入廣告後台，點選廣告組合 → 編輯廣告
  • 找到「目的地 URL」或「網站 URL」欄位
  • 把 Portaly 連結換成你的商品頁網址（例如 yourshop.com/products/xxx）
  • 廣告目標改成「轉換」→ 優化事件選「購買」
⚠️ 注意：Portaly 是集合多個連結的頁面，適合 Instagram 首頁連結，但廣告應該直接連到商品頁，減少一個跳轉步驟，能明顯提升轉換率。
5

修改廣告系列目標

目前你的廣告可能是「流量」目標（只優化點擊），改成「銷售」目標後 Meta 會把廣告推給更可能購買的人。

建立廣告系列 → 目標選「銷售」→ 轉換位置選「網站」→ 優化事件選「購買」
  • 新的 Pixel 前期資料不足，建議先讓它跑 7–14 天學習期
  • 學習期間預算不要大幅更動，讓演算法穩定收集資料
  • 每天至少 1 次購買轉換，演算法才能快速學習（預算參考：每日 NT$300–500 起）
